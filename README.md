# 📊 YouTube Channel Compare

YouTube 채널을 비교 분석하는 웹 애플리케이션입니다. 최대 5개의 채널을 동시에 비교하고, 다양한 지표를 시각화하여 채널의 강점과 약점을 한눈에 파악할 수 있습니다.

## ✨ 주요 기능

### 📈 종합 분석
- **채널 종합 점수**: 5가지 지표(구독자 규모, 조회수 파워, 성장성, 참여도, 활동성)를 기반으로 0~100점 평가
- **핵심 지표 비교**: 구독자, 조회수, 영상 수, 평균 조회수, 성장률, 업로드 주기 등
- **필터링**: 전체 영상 / 숏폼(≤3분) / 롱폼(>3분) 별도 분석

### 📊 시각화 차트
- **채널 밸런스 (레이더 차트)**: 5가지 평가 항목의 균형 확인
- **영상 길이 vs 조회수**: 최적의 영상 길이 파악
- **채널 성장 추세**: 최근 30일간의 조회수 변화
- **업로드 패턴**: 요일/시간대별 업로드 빈도 히트맵

### 🔍 세부 정보
- **주요 키워드**: 영상 제목에서 추출한 빈출 키워드 (클릭 시 YouTube 검색)
- **최근 영상 비교**: 최근 5개 영상의 상세 정보 및 썸네일
- **인터랙티브**: 차트 포인트 클릭 시 해당 영상/채널로 이동

## 🚀 사용 방법

### 1. YouTube Data API 키 발급
1. [Google Cloud Console](https://console.cloud.google.com/) 접속
2. 새 프로젝트 생성
3. "API 및 서비스" > "라이브러리" > "YouTube Data API v3" 활성화
4. "사용자 인증 정보" > "API 키 만들기"
5. (권장) API 키 제한 설정:
   - HTTP 리퍼러 제한: `https://yourusername.github.io/*`
   - API 제한: YouTube Data API v3만 허용

### 2. 애플리케이션 사용
1. 웹 페이지 접속
2. 우측 상단 설정 버튼 클릭
3. YouTube API 키 입력 및 저장
4. 비교할 채널 이름 또는 URL 입력 (최대 5개)
5. "분석 시작하기" 클릭

## 📊 평가 기준 상세

### 1. 구독자 규모 (30%)
- 산출: (구독자 수 ÷ 1,000,000) × 100
- 기준: 100만 구독자 = 100점

### 2. 조회수 파워 (30%)
- 산출: (평균 조회수 ÷ 500,000) × 100
- 기준: 최근 30개 영상 평균, 50만 조회수 = 100점

### 3. 성장성 (20%)
- 산출: 50 + 조회수 등락률
- 계산: 최근 15개 vs 이전 15개 영상 평균 비교
- 예시: 50% 증가 = 100점, 변동 없음 = 50점

### 4. 참여도 (10%)
- 산출: (평균 좋아요 비율 ÷ 0.05) × 100
- 계산: (좋아요 수 ÷ 조회수) 평균
- 기준: 5% 비율 = 100점

### 5. 활동성 (10%)
- 산출: 100 - (평균 업로드 주기 × 3.3)
- 예시: 매일 업로드 ≈ 97점, 3일 1개 = 90점
