채널 비교 기능은 정말 유용한 기능이 될 것 같습니다! 
최대 5개 채널을 한눈에 비교할 때, 단순히 숫자만 나열하는 것보다 **시각적이고 인사이트를 주는 기능**들이 들어가면 훨씬 매력적인 서비스가 될 것입니다.

다음과 같은 기능 아이디어들을 제안드립니다.

### 1. 📊 핵심 지표 비교 (기본 기능)
가장 기본적이지만 필수적인 데이터들을 깔끔한 테이블로 보여줍니다.
- **기본 스펙:** 구독자 수, 총 조회수, 동영상 개수, 채널 개설일
- **성과 지표:** 평균 조회수 (최근 30개 영상 기준), 구독자 대비 조회수 비율 (활성도)
- **업로드 주기:** 주당 평균 업로드 횟수

### 2. 🕸️ 채널 밸런스 차트 (추천 ⭐)
각 채널의 강점을 한눈에 비교할 수 있는 **육각형 레이더 차트 (Radar Chart)**입니다.
- **평가 항목:** `구독자 규모`, `조회수 파워`, `성실성(업로드 빈도)`, `영상 수`, `최근 성장세`
- 5개 채널을 겹쳐서 보여주면 어떤 채널이 어느 분야에서 우세한지 바로 알 수 있습니다.

### 3. 🤖 AI 채널 분석 (Gemini 활용)
단순 숫자가 아닌, **채널의 성격과 전략**을 AI가 분석해줍니다.
- **한 줄 요약:** "A채널은 '교육적'인 반면, B채널은 '예능형' 콘텐츠에 집중합니다."
- **주요 키워드 비교:** 각 채널이 자주 사용하는 태그나 제목 키워드 비교 (워드 클라우드)
- **타겟 시청자 추정:** "초보자 타겟" vs "전문가 타겟"

### 4. 📅 업로드 패턴 히트맵
경쟁 채널들이 **언제 영상을 올리는지** 시각화합니다.
- 요일별/시간대별 업로드 분포도
- *"경쟁자들이 쉬는 일요일 오후 2시에 올리세요!"* 같은 빈틈 공략 팁 제공

### 5. 🏆 대표 영상 1:1 매치
각 채널에서 **가장 성과가 좋았던 영상(Best Performance)**을 나란히 보여줍니다.
- 썸네일과 제목을 비교하여 벤치마킹할 포인트를 찾을 수 있습니다.

## 📖 사용 가이드

### 1️⃣ **API 키 설정**
- 우측 상단의 **"API 키 설정"** 버튼 클릭
- YouTube Data API v3 키를 입력하고 저장

### 2️⃣ **채널 검색 및 추가**
- 검색창에 채널명 입력 (예: "침착맨", "동네형", "빠더너스")
- 최대 5개 채널까지 선택 가능
- 잘못 추가한 채널은 X 버튼으로 제거

### 3️⃣ **분석 시작**
- 최소 2개 채널 선택 후 **"분석 시작하기"** 버튼 클릭
- 5-10초 정도 데이터 수집 시간 소요

### 4️⃣ **결과 확인**
- **핵심 지표 테이블**: 숫자로 직관적 비교
- **레이더 차트**: 5가지 항목 상대 평가
- **히트맵**: 채널별 업로드 시간 패턴
- **대표 영상**: 클릭하면 유튜브로 이동
